# 06_WEB_LANDING

Directorio oficial para lineamientos, contenido, estructura y evolución de la web de **Nexiom Intelligence Group**.

## Archivo maestro

El archivo principal de esta carpeta es:

```text
NEXIOM_WEB_LINEAMIENTOS.md
```

Este documento contiene las instrucciones permanentes que rigen el desarrollo, contenido, diseño, narrativa, formularios, landing pages, repositorios digitales, interfaces públicas y futuras integraciones de la web de Nexiom.

## Principios obligatorios para la web

Toda actualización de la web debe respetar:

1. Identidad corporativa de Nexiom Intelligence Group.
2. Narrativa premium, ejecutiva, tecnológica y confiable.
3. Privacidad, confidencialidad y protección de datos.
4. Cumplimiento normativo aplicable.
5. Fuentes internas aprobadas del proyecto.
6. Academia online, servicios comerciales y captación de clientes.
7. Atención clara a clientes, participantes, aliados y colaboradores.
8. Portabilidad, respaldo y trazabilidad documental.
9. Mejora continua, escalabilidad e innovación.
