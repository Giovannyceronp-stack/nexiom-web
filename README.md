# Nexiom Web + Academia

Sitio público de Nexiom Intelligence Group con Academia integrada y curso demo **Excel Básico Administrativo**.

## Incluye

- Landing pública de Nexiom.
- Identidad corporativa en azul profundo, bronce y blanco.
- Academia Nexiom dentro de la web.
- Acceso demo con 3 datos de registro.
- Curso Excel Básico Administrativo de 30 minutos.
- Evaluación automática.
- Constancia demo imprimible.
- Descarga CSV demo para CRM académico.
- Nexi, asistente virtual persistente.
- Workflow de GitHub Pages sin costo adicional.

## Seguridad

Esta versión es una demostración estática. No envía datos reales a internet. Para producción se requiere backend seguro, base de datos, autenticación real, aviso de privacidad y control de acceso por roles.

## Publicación

El workflow `.github/workflows/deploy-pages.yml` publica el sitio con GitHub Pages cuando Pages esté configurado con **Source: GitHub Actions**.
